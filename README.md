# EuroSkills 2027 – National Selection (Round 2)

This repository contains my solution for the **EuroSkills 2027** National Selection (Round 2). This project successfully secured my place in the **National Finals**.

## Overview
While the core of this solution was built during the competition, this version includes post-contest refinements. After qualifying for the finals, I refactored the code to fix minor bugs and polish the implementation that couldn't be perfected within the strict competition time limits.

## Highlights
* **Competition:** EuroSkills 2027 National Selection
* **Result:** Qualified for the National Finals 🏆
* **Status:** Post-competition refined & bug-fixed version
